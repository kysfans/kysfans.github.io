CMD(Command Prompt)是 Windows 作業系統中的命令行界面，它允許使用者通過輸入指令來執行各種作業系統相關的任務和操作。
以下是一些常用的 CMD 基本指令：

盤符:

切換盤符

例如：

F:

切換到F盤。

dir
列出當前目錄中的檔案和子目錄。

md
mkdir

建立新目錄。例如：md NewFolder 將在當前目錄中創建一個名為 NewFolder 的新目錄。

rd 或 rmdir

刪除目錄。例如：rd OldFolder 將刪除當前目錄中名為 OldFolder 的目錄。

copy

複製檔案

例如：

copy File1.txt File2.txt

將 File1.txt 複製到 File2.txt。

move

移動或重新命名檔案或目錄。

例如：

move OldFile.txt NewFile.txt

將 OldFile.txt 重命名為 NewFile.txt。

del
erase

刪除檔案

例如：

del UnwantedFile.txt

將刪除名為 UnwantedFile.txt 的檔案。

cls

清空命令提示字元窗口上的內容。

exit

結束命令提示字元。

ping

用於測試與指定主機的連接。

ipconfig

顯示網絡設定信息，例如 IP 位址、子網路遮罩和預設閘道。

tasklist

列出正在運行的程序和任務。

taskkill

終止正在運行的程序。例如：taskkill /IM notepad.exe 將終止所有名為 notepad.exe 的 Notepad 程序。

help

來查看更多可用的指令列表

<指令> /?

來查看每個指令的詳細說明和用法。
