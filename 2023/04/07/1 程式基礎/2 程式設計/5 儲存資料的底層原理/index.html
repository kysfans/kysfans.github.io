儲存資料的底層原理

2023-04-07 程序設計 483 個字

電腦是二進位。
電腦儲存資料的最小單位是1byte (位元組)
1TB = 1024GB, 1GB = 1024MB, 1MB = 1024KB, 1KB = 1024byte,1 byte = 8 bit ,1 bit = 1個二進位單位
如：在電腦中，二進位的110以00000110的方式儲存。

一、數字
1，二進位：只有0,1，逢2進1。

2，十進位：只有1到9，逢10進1。

(i)十進位換算二進位，使用除二整除法，例如：

6 / 2 = 3 餘 0
3 / 2 = 1 餘 1

所以 6 在二進位的表達是 110。

(ii)二進位換算十進位

例如： 0011 = 2的3次方 x 0 + 2的2次方 x 0 + 2的1次方 x 1 + 2的0次方 x 1 = 3

3，八進制：只有1到7，逢8進1。
(i)以二進位表示八進制的單位
如：001 = 1, 010 = 2, 011 = 3, 100 = 4, 101 = 5, 110 = 6, 111 = 7

4，十六進位
以4位元二進位表示十六進位的單位
如：0001 = 1, 0010 = 2, 0011 = 3, 0100 = 4, 0101 = 5, 0110 = 6, 0111 = 7, 1000 = 8, 1001=9, 1010 = A, 1011= B, 1100 = C, 1101 = D, 1110 = E, 1111 = F

5，注意事項
Java 支援書寫二進位，八進制、十六進位。
二進位：0B or 0b
八進制：0
十六進位：0X or 0x

二、文字
美國資訊交換標準編碼(ASCII)：規定了現代英文，數位，和其他西歐字元對應的數位。再把數位換算成2進制方式儲存。

如：’a’ = 97
當執行程式是，字元會輸出對應的文字。不過當字元參與到運算的話，程式會先轉換成對應的ASCII 再進行運算。

三、圖像
圖像由一個一個象素點組成，每一個圖元點可以表達成：(0255, 0255, 0~255)的方式。再轉以2進制儲存。

四、聲音
把聲形的波形用2進製表達和儲存。
