Java List 遍歷

2023-04-23 Java其他概念 597 個字
List

在 Java 中，你可以使用多種方式來遍歷 List 中的元素，包括使用增強的 for 迴圈、使用迭代器和使用傳統的 for 迴圈。以下是這些方法的詳細說明：

  1. 使用增強的 for 迴圈（Enhanced for loop）：
    增強的 for 迴圈是一種簡潔的方式來遍歷集合，特別適合於只需要遍歷元素而不需要修改集合的情況。增強的 for 迴圈適用於所有實現了 Iterable 接口的集合，包括 List。
import java.util.ArrayList;  
import java.util.List;  
  
public class Main {  
    public static void main(String[] args) {  
        List<String> names = new ArrayList<>();  
        names.add("Alice");  
        names.add("Bob");  
        names.add("Carol");  
  
        // 使用增強的 for 迴圈遍歷 List 中的元素  
        for (String name : names) {  
            System.out.println(name);  
        }  
    }  
}
  1. 使用迭代器（Iterator）：
    如果需要在遍歷時修改集合或手動控制遍歷的過程，可以使用迭代器來遍歷 List。迭代器提供了遍歷集合的一種方式，並且支持在遍歷時進行修改操作。
import java.util.ArrayList;  
import java.util.Iterator;  
import java.util.List;  
  
public class Main {  
    public static void main(String[] args) {  
        List<String> names = new ArrayList<>();  
        names.add("Alice");  
        names.add("Bob");  
        names.add("Carol");  
  
        // 使用迭代器遍歷 List 中的元素  
        Iterator<String> iterator = names.iterator();  
        while (iterator.hasNext()) {  
            String name = iterator.next();  
            System.out.println(name);  
        }  
    }  
}
  1. 使用傳統的 for 迴圈：
    你也可以使用傳統的 for 迴圈通過索引來遍歷 List。這種方式可以根據索引來訪問元素，並進行一些複雜的操作。
import java.util.ArrayList;  
import java.util.List;  
  
public class Main {  
    public static void main(String[] args) {  
        List<String> names = new ArrayList<>();  
        names.add("Alice");  
        names.add("Bob");  
        names.add("Carol");  
  
        // 使用傳統的 for 迴圈遍歷 List 中的元素  
        for (int i = 0; i < names.size(); i++) {  
            String name = names.get(i);  
            System.out.println(name);  
        }  
    }  
}

以上是三種常見的遍歷 List 的方式。你可以根據實際需求來選擇適合的遍歷方法。如果只需要遍歷元素，且不需要修改集合，則增強的 for 迴圈是一個簡潔和方便的選擇。如果需要在遍歷時修改集合，則應使用迭代器。而傳統的 for 迴圈則提供了更多的控制，適合在特定場景下使用。
