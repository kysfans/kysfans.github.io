3 Mybatis 配置

2023-09-04 ORM框架 940 個字
筆記

配置 MyBatis、創建映射文件並建立 Java 對象和數據庫表之間的映射關係需要進行一系列步驟。以下是一個基本的指南，幫助您開始使用 MyBatis 進行數據庫操作：

步驟一：配置 MyBatis

引入 MyBatis 相關的依賴： 在您的項目中，首先需要將 MyBatis 相關的依賴加入到項目的 build 配置中，以便您可以在代碼中使用 MyBatis。

org.mybatis.spring.boot mybatis-spring-boot-starter 2.2.0

配置数据源：在application.properties或application.yml文件中配置您的数据源信息，包括数据库URL、用户名、密码等。

spring.datasource.url=jdbc:mysql://localhost:3306/mydatabase
spring.datasource.username=myuser
spring.datasource.password=mypassword
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver

配置MyBatis：在application.properties或application.yml文件中添加MyBatis的配置信息。

mybatis.mapper-locations=classpath:mapper/*.xml

步驟二：創建映射文件

定義映射器接口： 創建一個 Java 接口，該接口將定義與數據庫操作相關的方法。每個方法將對應一個 SQL 查詢或更新操作。

@Mapper
public interface UserMapper {
List getAllUsers();
// 其他方法…
}

創建映射文件： 創建一個 XML 文件，這個文件將包含 SQL 語句和與 Java 方法的映射。這些映射文件通常與映射器接口相對應，並存儲在一個特定的目錄中。

  1. 在Service中使用Mapper：在您的Service层中注入Mapper接口，并在业务方法中调用Mapper的方法。

@Service
public class UserService {
private final UserMapper userMapper; 

@Autowired  
public UserService(UserMapper userMapper) {  
    this.userMapper = userMapper;  
}  

public List<User> getAllUsers() {  
    return userMapper.getAllUsers();  
}

}

在 MyBatis 中，映射文件（Mapper XML 文件）用於定義 SQL 查詢、映射關係和其他與數據庫操作相關的內容。以下是創建 MyBatis 映射文件的基本步驟：

步驟一：創建映射文件

  1. 在您的 MyBatis 項目中，創建一個文件夾（通常命名為 mapper 或 mappers）來存放映射文件。

  2. 在該文件夾中創建一個 XML 文件，命名為與您的映射器接口相對應的名稱，並以 .xml 為後綴。例如，如果您有一個名為 UserMapper 的映射器接口，則映射文件可以命名為 UserMapper.xml。

步驟二：定義命名空間

在映射文件的頂部，定義命名空間（namespace），該命名空間應該與您的映射器接口的全限定名相同。 

<!-- UserMapper.xml -->  
<mapper namespace="com.example.UserMapper">  
    <!-- 映射操作將在這裡定義 -->  
</mapper>

步驟三：定義 SQL 查詢和操作

在 <mapper> 元素中，您可以定義多個 <select>、<insert>、<update> 和 <delete> 元素，分別對應不同的 SQL 操作。 

<!-- UserMapper.xml -->  
<mapper namespace="com.example.UserMapper">  
    <!-- 定義一個查詢所有用戶的 SQL -->  
    <select id="getAllUsers" resultType="User">  
        SELECT * FROM users  
    </select>  
      
    <!-- 定義一個按照 ID 查詢用戶的 SQL -->  
    <select id="getUserById" resultType="User" parameterType="Long">  
        SELECT * FROM users WHERE id = #{id}  
    </select>  
      
    <!-- 定義一個插入用戶的 SQL -->  
    <insert id="insertUser" parameterType="User">  
        INSERT INTO users (id, username, email) VALUES (#{id}, #{username}, #{email})  
    </insert>  
      
    <!-- 其他操作... -->  
</mapper>

步驟四：映射結果

在 <select> 元素中，使用 <resultMap> 元素定義結果集的映射。 

<!-- UserMapper.xml -->  
<mapper namespace="com.example.UserMapper">  
    <resultMap id="userResultMap" type="User">  
        <id property="id" column="id"/>  
        <result property="username" column="username"/>  
        <result property="email" column="email"/>  
    </resultMap>  
      
    <select id="getUserById" resultMap="userResultMap" parameterType="Long">  
        SELECT * FROM users WHERE id = #{id}  
    </select>  
      
    <!-- 其他操作... -->  
</mapper>
