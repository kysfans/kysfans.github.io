貳、JAVA 基礎語法

一、基本概念
一個 Java 程式可以認為是一系列物件的集合，而這些物件通過調用彼此的方法來協同工作。
下面將簡要介紹下類、物件、方法和個體變數的概念。

1，對象：物件是類的一個實例，有狀態和行為。

例如，一條狗是一個物件，它的狀態有：顏色、名字、品種；行為有：搖尾巴、叫、吃等。

2，類：類是一個範本，它描述一類物件的行為和狀態。
3，方法：方法就是行為，一個類可以有很多方法。邏輯運算、資料修改以及所有動作都是在方法中完成的。
4，執行個體變數：每個物件都有獨特的執行個體變數，物件的狀態由這些執行個體變數的值決定。

二、編寫 Java 程式的注意事項：

1，原始檔案名：原始檔案名必須和類名相同。當保存檔的時候，你應該使用類名作為檔案名保存（切記 Java 是大小寫敏感的），檔案名的尾碼為 .java。（如果檔案名和類名不相同則會導致編譯錯誤）。
2，主方法入口：所有的 Java 程式由 public static void main(String[] args) 方法開始執行。

三、JAVA 專案結構

1，項目(project)
2，模組(module)
3，包(package)

用來分門管理不同的類，有助管理和維護程式

語法：package 公司名 技術名稱
建議有意義的英文小寫

四，專案結構的注意事項
1，必須寫在檔第一行
2，同一包下的類可以直接存取
3，不同包的類需要導包

格式import 包名.類名

4，假如一個類需要用到相同名字的類，第二個類需要帶包名存取。

五、JAVA 注釋
注釋是不會被編譯和執行的內容，僅供工程師閱讀和參考用。JAVA的注釋分為三種，分別是：單行注釋、多行注釋和文檔注釋。

1，單行注釋
以雙斜杠“//”標識，只能注釋一行內容，用在注釋資訊內容少的地方。

// 單行注釋

2，多行注釋
包含在“/”和“/”之間，能注釋很多行的內容。為了可讀性比較好，一般首行和尾行不寫注釋資訊。

/* 多行
 * 注釋
 */

3，文檔注釋
包含在“/*”和“/”之間，也能注釋多行內容，一般用在類、方法和變數上面，用來描述其作用。注釋後，滑鼠放在類和變數上面會自動顯示出我們注釋的內容。文檔注釋的內容會被提取到程式說明書中。

注意：文檔注釋能嵌套單行注釋，不能嵌套多行注釋和文檔注釋，一般首行和尾行也不寫注釋資訊。

/**
 * 就象
 * 這樣
 */

六、變數
變數存儲一個資料的記憶體區域，而且裡面存儲的資料可以變化。

變數類型：資料在程式中的書寫格式。

基本資料類型

整數

1，int
存儲需求：4位元組4
資料範圍：-217483648~-217483647

2， short
存儲需求：2位元組
資料範圍：-32768~32767

3，long
存儲需求：8位元組8
資料範圍：-9223372036854775808~-9223372036854775807

4，byte
存儲需求：1位元組
資料範圍：-128~127

(2.1.1.2) 浮點(小數)
1，float
存儲需求：4位元組
資料範圍：+-3.40282347E+38F (有效位數6-7位)
2，double
存儲需求：8位元組
資料範圍：+-1.79769313486231570E+308(有效位數15位)

(2.1.1.3) 字元(char )
存儲需求：1位元組
資料範圍：如’A’, ‘0’, ‘我’
必須使用單引號。

(2.1.1.4) 布林值(Boolean)
資料範圍：true, false
 
(2.1.2) 引用資料類型
(2.1.2.1) 字串
資料範圍：如”hello world”。
必須使用雙引號，內容可有可無。

(2.1.1.2)空值
資料範圍：null

(2.2) 變數定義的格式
資料類型 變數名稱 = 初始值;

(2.2.1) 修改變數的值
(2.2.1.1) 方法1
資料類型 變數名稱 = 初始值;
變數名稱 = 變數名稱 + 值;
(2.2.1.2) 重新賦值
資料類型 變數名稱 = 初始值;
變數名稱 = 重新賦值;

(2.2.2) 變數的注意事項
1，變數必須先聲名類型再使用。
2，聲名變數類型後，不能儲存其他類型的資料。
3， 變數聲名的有效範圍是從定義開始到”}”結束1，同一個範圍內不能定義2個同名的變數。
4， 變數定義的時候可以沒有初始值，但是變數使用是必須有初始值。

(2.3) 類型轉換
(2.3.1) 自動類型轉換
類型範圍小的類型可以自動轉換範圍大的類型
如： byte a = 20; int a = b;
(2.3.1.1) 自動類型轉換的底層原理
a = 00001100, b= 00000000 00000000 00000000 00001100

(2.3.1.2) 運算式的自動類型轉換
運算式的最終結果類型是由運算式中的最高類型決定。
為了避免運算後超出範圍，在運算式中，byte, short, char是直接轉換成 int。
(2.3.3) 強制類型轉換
資料類型 變數名稱2 = (資料類型) 變數名稱;
(2.3.3.1) 自動類型轉換的底層原理
b= 00000000 00000000 00000000 00001100, a = 00001100
如果超出範圍會造成資料丟失或資料溢出
小數轉換整數會丟失所有小數項。
如：byte 超出範圍會變為二進位補數，也就是負數

 
(3) 儲存資料的底層原理
電腦是二進位。
電腦儲存資料的最小單位是1byte (位元組)
1TB = 1024GB, 1GB = 1024MB, 1MB = 1024KB, 1KB = 1024byte,1 byte = 8 bit ,1 bit = 1個二進位單位
如：在電腦中，二進位的110以00000110的方式儲存。

(3.1) 數字
(3.1.1)二進位
二進位的意思是：只有0,1，而且逄2進1。

(3.1.1.1)十進位換算二進位
除二整除法
如：
6 / 2 = 3 餘 0
3/2 = 1 餘 1
所以 6 在二進位的表達是 110。

(3.1.1.2) 二進位換算十進位
如： 0011 = 23 x 0 + 22 x 0 + 21 x 1 +20 x 1 = 3

(3.1.2) 八進制
以3位元二進位表示八進制的單位
如：001 = 1, 010 = 2, 011 = 3, 100 = 4, 101 = 5, 110 = 6, 111 = 7

(3.1.3) 十六進位
以4位元二進位表示十六進位的單位
如：0001 = 1, 0010 = 2, 0011 = 3, 0100 = 4, 0101 = 5, 0110 = 6, 0111 = 7, 1000 = 8, 1001=9, 1010 = A, 1011= B, 1100 = C, 1101 = D, 1110 = E, 1111 = F

(3.14) 注意事項
Java 支援書寫二進位，八進制、十六進位。
開頭
二進位：0B or 0b
八進制：0
十六進位：0X or 0x
 
(3.2) 文字
美國資訊交換標準編碼(ASCII)：規定了現代英文，數位，和其他西歐字元對應的數位。再把數位換算成2進制方式儲存。

如：’a’ = 97
當執行程式是，字元會輸出對應的文字。不過當字元參與到運算的話，程式會先轉換成對應的ASCII 再進行運算。

(3.3) 圖像
圖像由一個一個象素點組成，每一個圖元點可以表達成：(0255, 0255, 0~255)的方式。再轉以2進制儲存。

(3.4) 聲音
把聲形的波形用2進製表達和儲存。

(3.5) 影像
圖片加聲音  

(4) 不能作為變數名稱的字
(4.1) 關鍵字
JAVA保留的特殊功能的單詞，如：public, class, byte等
(4.2) 識別字
由一些字元和符號組合的名稱，用於給類，方法，變數等起名字的規矩
(4.2.1)識別字命名規範
基本要求：由數位、字母、底線和美元符等組成
強制要求：不能以數位開頭，不能是關鍵字，區分大小字
變數名稱：滿足識別字的要求，建議有意義的全英文，首字母小寫，滿足駝峰規則。
類名稱：滿足識別字的要求，建議有意義的全英文，首字母大寫，滿足駝峰規則。

(7) 陣列
陣列：是引用資料類型，用來儲存一批同類型的記憶體區域，可被理解為容器，也就是儲存在記憶體中的位址資訊。

靜態初始化陣列：
Int [] arr = {20,10};
String [] name = {“fuckyou”, “Fuck you”};

動態初始化陣列：
只定義陣列的類型和長度
例如：
Int [] arr = new int[3]
Arr [0] = 10

存取陣列成員
System.out.println(Arr[1])
存取陣列長度
System.out.println(Arr.length)
最大索引值
System.out.println(Arr.length-1)

陣列預設值：
整數 = 0
小數 = 0.0
Boolean = false
引用資料類型 = null

陣列遍歷
int arr[] = {20,10,30};
for (int i = 0; i < arr.length; i++) {
System.out.println(arr[i]);
}

(6)流程控制
(6.1)分支
(6.1.1) if
格式：

if (condition){
    Statement 
}

else If (condition){
    Statement
}

else (condition){
    Statement
}

適合區間匹配

(6.1.2) switch

Switch (condition){
    Case value 1:
        Statement;
        Break;
        Default:
        Statement;
}

適合值匹配
注意事項：
1， condition只能是byte,short,int,char.string和枚舉(enum)
2， case不能是變數
3， 不寫break會造成穿透，意思是說不會進行任何匹配就直接執行下一個case的statements。

(6.2)迴圈
(6.2.1) For
For ：用已知的迴圈次數求結果for，控制變數不可重複使用

格式：

For (初始化語句; 迴圈條件; 反覆運算語句{
    Statements;
}

例子：

For(int i= 0; i<3; i++){
    System.out.println(“hi”)
}

 

(6.2.2) while
While：用已知的結果求迴圈次數，控制變數可重複使用
格式：
例子：

int I = 0
while (i<3){
System.out.println(“hello”);
I++; 
}

(6.2.3) do-while
do-while：至少執行一次迴圈
格式：
例子：

Int I = 0;
Do{
    System.out.println(“hello”);
}
while(i<=3);

(6.2.4) 閉環
閉環：不會停止的迴圈。
例子(1)：

    for (;;){
        System.out.println("h");
    }

例子(2)：

    do {
        System.out.println("h");
    }
    while (true);

例子(3):
while (true){
System.out.println(“h”);
}

(6.2.5) 迴圈嵌套
迴圈嵌套：迴圈中有另一個迴圈。先執行內部迴圈才執行外部迴圈。
例子：

for (int i = 0; i <3 ; i++) {
    for (int j = 0; j < 3; j++) {
        System.out.println(i);
        System.out.println(j);
    }
}

(6.2.6)控制關鍵字
Break 跳出並結束迴圈
Continue 跳出當次迴圈進入下次迴圈 Skip 

(8) 方法
方法：把一段程式碼封裝成一個功能，方便重複調用。
好處：提高程式碼複用性，邏輯更清晰。
定義方法
格式：
修飾詞 返回數值型別 方法名 (形參列表){
需要執行的程式碼;
Return 返回值;
}

例子：

public static int add (int a, int b){
    int c = a + b;
    return c;

調用方法：
格式：
方法名(實參)
例子：

int rs = add(100,200);

注意事項：
如有返回數值型別 必須使用return 返回返回數值型別的資料。
可以有0~N個形參。
不需返回結果，返回數值型別為void 不需return。
方法順序不重要
方法不能嵌套。
Return 後不能再加語句，因為不會執行。
方法只有在調用才會執行
設置變數接收返回值。
 
傳遞機制：
傳輸實參給形參時，是傳遞值。
基本類型: 數據值
參考類型：地址值

值傳遞是指在調用函數時將實際參數複製一份到函數中，這樣的話如果函數對其傳遞過來的形式參數進行修改，將不會影響到實際參數。
引用傳遞是指在調用函數時將物件的位址直接傳遞到函數中，如果在對形式參數進行修改，將影響到實際參數的值。
